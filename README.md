## Setup

#### Install dependencies
```
brew install python3 poetry
```

#### Build project
```bash
poetry install --no-root
playwright install
```

#### Running from CLI (no IDE)
To avoid conflicts between project lib and system libs
```
python3 -m venv .venv
source .venv/bin/activate
```
